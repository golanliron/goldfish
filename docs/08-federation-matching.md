## DNA Matching Algorithm (Pseudocode)
```
function matchFederation(org, federation):
  score = 0

  // Population overlap
  for pop in org.populations:
    if pop in federation.funded_populations:
      score += 20

  // Domain overlap
  for domain in org.domains:
    if domain in federation.funded_domains:
      score += 15

  // Geographic match (P2G)
  if org.region in federation.p2g_regions:
    score += 25  // Warmest entry point

  // Similar org funded before
  for funded_org in federation.funded_organizations:
    if similar(org, funded_org):
      score += 30  // Strongest signal

  // Emergency/rebuild alignment
  if org.does_trauma OR org.does_resilience:
    score += 10  // Current priority

  return normalize(score, 0, 100)
```
